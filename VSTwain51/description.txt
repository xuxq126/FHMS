
VintaSoftTwain ActiveX Control will help you to control scanners, digital or
web cameras or any other TWAIN devices. You can fully control the image
acquisition process, use the automatic document feeder, clean up images using
noise removal, auto border crop, blank page detection, save acquired images
to disk, SQL server or upload them to the server as BMP, JPEG or TIFF files.
ActiveX can be used in your programs written on any programming languages
which support work with ActiveX/COM controls.
ActiveX was tested and has examples of use in: MS VB.NET, MS C#, MS VB 6.0,
MS VC++ 6.0, Borland Delphi 7.0, Borland C++ Builder 6.0, Borland Delphi .NET,
Borland C# Builder, Visual FoxPro 8.0, Gupta SQL Windows 3.1, MS Access,
ASP.NET, VBScript, JavaScript, MS Excel, MS Word, IBM Lotus Notes,
IBM Object Rexx language.
